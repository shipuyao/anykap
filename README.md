anykap - capture things in k8s with your tools, when it matters
===============================================================

Installation
------------

Currently anykap is in pre-Alpha. To install the dev version for testing with a 
recent Python3:

``` shell
python3 -m venv .venv
. .venv/bin/activate
pip install -U git+https://github.com/tdihp/anykap.git@main
```
